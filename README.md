# OW - Sitio Web Dinámico

Bienvenido al repositorio oficial de **Only Way**, una plataforma web diseñada para representar la identidad, valores y catálogo visual de la marca de ropa deportiva.

## 🎯 Objetivo del Proyecto
El sitio funciona como el núcleo digital de **Only Way**, ofreciendo una experiencia de usuario fluida y moderna. Está enfocado en transmitir la esencia del *sportswear* de alta calidad y facilitar canales de comunicación directos con los clientes.

---

## 🚀 Tecnologías Utilizadas

El proyecto aprovecha un stack moderno para garantizar rendimiento y escalabilidad:

* **Frontend:** HTML5, JavaScript (ES6+) y [Tailwind CSS](https://tailwindcss.com/) para una interfaz altamente responsiva y estilizada.
* **Backend:** [Node.js](https://nodejs.org/) con Express para la gestión de rutas y lógica de servidor.
* **APIs Internas:** Consumo de rutas de API personalizadas para la carga dinámica de información de la tienda.
* **Comunicación:** Sistema de **Mail Sender** integrado para la gestión de formularios y contacto directo.
* **Despliegue:** [Netlify](https://www.netlify.com/) (Frontend) y configuración de funciones serverless o backend dedicado.

---
## Características Principales
- **Navegación Multi-página:** Estructura organizada en múltiples archivos HTML para una mejor jerarquía de información.
- **Reseñas Dinámicas:** Los datos de las reseñas no están "quemados" en el HTML; se obtienen mediante peticiones a una API, facilitando futuras actualizaciones.
- **Servicio de Mensajería:** Integración de un módulo de envío de correos electrónicos para procesar consultas de clientes de forma automática.
- **Optimización Tailwind:** Uso de JIT (Just-In-Time) compiler para un rendimiento superior en el renderizado de estilos.
---
## 🌐 Arquitectura de Despliegue
El sitio utiliza el flujo de **Integración Continua (CI/CD)** de Netlify. La lógica del servidor y el envío de correos están optimizados para funcionar de manera eficiente en entornos de producción.
